OnePayment

Base URL: /v1, Version: 1.0.0

Here you will find the documentation for the OnePayment API.

Schemes: https, http

Summary

Tag: user

Everything about your users.

Operation Description
GET /user/{id}

Gets all information relating to a user.
POST /user/login

Logs user into the system
POST /user/signup

Allows a user to sign up.

Tag: ledger

Everything about ledgers.

Operation Description
POST /ledger

Add a new ledger to the system.
GET /ledger/{id}

Get all ledger information by ledger id.
DELETE /ledger/{id}

Deletes a ledger with the provided id
DELETE /transaction/{id}

Deletes a transaction with the provided id

Tag: transaction

Information relating to transactions in a ledger.

Operation Description
GET /transaction/{id}

Get all transaction information by transaction id.

Operation Description
